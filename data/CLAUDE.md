# Data Directory

Generated and runtime data files.

## Subdirectories

### manifests/
Section manifest JSON files generated by preprocessing. Each file corresponds to one URL and contains:
- `source_url` - Original URL
- `source_name` - Human-readable name
- `category` - Category for search weights
- `sections` - List of extracted sections with id, title, description, keywords, anchor_heading

### vectordb/
ChromaDB persistence directory and metadata:
- `chroma.sqlite3` - ChromaDB database
- `metadata_only.json` - Sections with `semantic_weight: 0` (spells, archetypes)

## Regenerating Data

```bash
# Regenerate manifests
poetry run pfrules-preprocess --force

# Rebuild vector index
poetry run pfrules-vectordb --build
```

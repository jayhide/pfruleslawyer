# Tools Directory

Standalone utility scripts not part of the main application.

## Subdirectories

### spell_sorting/
Utility for comparing spell lists between classes.

- `compare_spells.py` - Script to compare spell availability
- `druid_spells.txt` - Druid spell list
- `wizard_spells.txt` - Wizard spell list

Used for game analysis, not part of the RAG pipeline.
